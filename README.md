# Aura AI Chat 🌌

A clean, modern, and immersive chat interface powered by **Gemini 3 Flash**. Designed for speed, privacy, and aesthetic clarity.

[Русская версия ниже](#aura-ai-chat-ru)

## Features
- 🚀 **Free Access**: Uses Gemini 3 Flash without requiring a direct billing setup for the end-user (powered by Google AI Studio).
- 🎨 **Immersive UI**: Dark, atmospheric design with fluid animations.
- 📝 **Markdown Support**: Full support for code blocks, lists, and formatting.
- 📱 **Responsive**: Built for desktop and mobile devices.
- 🔒 **Privacy Focused**: No data tracking, local engine simulation.

## Tech Stack
- **React 19** + **TypeScript**
- **Vite** (Build tool)
- **Tailwind CSS 4** (Styling)
- **Motion** (Animations)
- **Lucide React** (Icons)
- **Google Generative AI SDK**

## Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Copy `.env.example` to `.env`:
   ```bash
   cp .env.example .env
   ```
4. Add your **GEMINI_API_KEY** in the `.env` file. You can get one for free at [Google AI Studio](https://aistudio.google.com/).

### Development
Run the development server:
```bash
npm run dev
```
